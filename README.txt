# globalization_asp_net_mvc
Demo showing how to implement globalization in ASP.NET MVC 
1. The resources files must be PUBLIC
2. You can organize the resources in folders
3. You need a filter to detect language
4. Add the filter in FilterConfig (App_Start/FilterConfig)
5. Change the route config (App_Start/RouteConfig)
6. Implement in your entities (Models/AccountViewModels)
7. Implement in your views (Shared/_Layout)
